## 🎮 1. 게임 개요

**게임 제목**: 공주와 비밀의 레시피 

**장르**: 어드벤처 / 아케이드  / 로그라이트

### **🎯 엘리베이터 피치**

 평범한 재료로는 결코 완성할 수 없는 환상의 케이크를 만들기 위해, 주인공인 공주가 스테이지의 기믹을 돌파하며 재료를 수집하는 탑뷰 어드벤처 게임.

### 🧩 핵심 메커니즘 3가지

1. **리스크 캐리 메커니즘:** 특정 아이템을 보스의 공격을 피해 목표 지점으로 안전하게 운반 → 성공 누적 시 스테이지 진척.
2. **패턴 변화:** 플레이어의 진척도가 일정 이상 누적 시 보스의 모습이 변화하며, 체력 구간에 따라 패턴이 가속.
3. **페널티 기반 상태 이상:** 플레이어가 피격 시 슬로우/기절 등 디버프 부여

## 🧁 2. 게임 스토리

**배경 설정**

요정 왕국의 먹보 공주는 어느 날 왕궁의 서고에서 금지된 환상의 케이크 레시피를 발견한다. 맛있는 케이크를 너무나도 좋아하는 공주는 환상의 케이크의 재료를 찾아 깊은 숲속으로 향한다.

**플레이어 목표**

공주의 방을 떠나 숲속으로 향해 재료를 구해야 한다. 플레이어는 매 스테이지에 진입하기 전 재료를 선택할 수 있으며, 선택한 재료에 따라 보스와 스테이지의 디자인이 변화한다. 빵, 초콜릿, 과일 스테이지를 모두 클리어하면 선택한 재료에 따라 상응하는 완성된 케이크 이미지와 클리어 타임이 확인 가능하고, 엔딩 컷씬이 재생된다.

**스토리 개요**

공주는 순수하고 밝으며 아기자기한 존재지만, 그녀가 마주하는 재료들은 전부 살아 움직이는 생명체다. 그녀는 환상의 케이크를 먹겠다는 순수한 목표로 숲속을 탐험하며 재료를 수집하지만, 공주를 막아서는 보스들의 반응과 엔딩의 컷씬을 통해 금지된 환상의 케이크는 사실 생명이 깃든 잔혹한 무언가였음을 암시하며 막을 내린다.

### 🕹 Stage 1 — 튜토리얼 (공주의 방)

**난이도:** 없음

**목표:** 기본 조작 익히기

**클리어 조건:** 지도와 상호작용해 다음 스테이지를 선택

**게임플레이:** 시작 컷씬(단편만화)이 재생된 후 공주의 방이 등장한다. 플레이어는 방바닥에 크게 그려진 조작법을 보고 조작을 익힐 수 있다. 방 상단의 지도와 상호작용하여 원하는 재료를 선택하면 스폰지케이크 또는 파운드케이크 스테이지로 넘어간다. 지도에는 다음 스테이지의 목표와 위험 요소가 간략히 표시되어 있다.

| **요약** |  |

### 🍞 Stage 2 — 빵 만들기 (스폰지 / 파운드 케이크)

**난이도:** 쉬움

**목표:** 밀가루 괴물에게 계란을 일정 횟수 먹이기

**클리어 조건:** 계란을 일정 횟수 이상 투입하면 자동으로 지도가 펼쳐지며 다음 스테이지 선택

**핵심 메커니즘:** 계란 보급, 좌우로 이동하는 닭 몬스터 피하기

**게임플레이:** 잔디밭 스테이지에 입장하면 밀가루 보스와 닭 몬스터들이 놀라며 움직이기 시작한다. 화면 하단의 둥지와 상호작용해 계란을 획득한다(최대 1개 소지). 플레이어는 공주를 조작해 계란을 얻고, 좌우로 왕복하는 닭 몬스터를 피해 화면 상단의 밀가루 괴물까지 도달한 뒤 상호작용으로 계란을 먹인다. 이후 다시 둥지로 이동해 계란을 획득하는 과정을 반복한다.

**적/위험 요소:** 플레이어는 4칸의 체력을 지닌다. 닭 몬스터와 접촉 시 체력이 한 칸 감소한다. 계란을 든 상태로 접촉하면 계란이 소실되며, 둥지에서 다시 보급해야 한다.

**밀가루 보스:** 눈과 입이 그려진 밀가루 봉투 형태다. 스폰지케이크를 선택한 경우 봉투 위에 원통 모양의 밝은색 빵이 얹어져 있고, 파운드케이크를 선택한 경우 직육면체 모양의 어두운 빵이 놓여 있다. 계란을 일정 횟수 이상 먹이면 빵빵해진 모습으로 변한다. 스테이지 클리어 시 봉투가 터지면서 공주에게 밀가루를 제공한다.

**닭 몬스터:** 계란의 주인들로, 계란을 훔쳐가는 공주에게 분노해 화난 표정을 짓고 있다. 계란을 일정 횟수 이상 먹이면 이동 속도가 빨라진다. 클리어 시 정지하며 우는 얼굴로 자리에 주저앉는다.

| **요약** |  |
| --- | --- |
| **핵심 메커닉** | 둥지에서 계란 보급 (최대 1개 보유 가능), 횡 이동하는 닭 몬스터 피하기. 계란을 보유한 채로 밀가루 괴물과 상호작용 시 계란을 먹임 |
| **진행 구조** | 일정 횟수 계란 먹일 시 자동 스테이지 클리어 |
| **위험 요소** | 닭과 접촉 시 체력 -1, 보유 중인 계란 소실 → 둥지에서 재보급 필요 |
| **보스 설정** | 밀가루 봉투 형태의 괴물. 스폰지(밝은색 원통형), 파운드(어두운 직육면체형)로 디자인 분기 |
| **레퍼런스** | 길건너 친구들 (Crossy Road): 리듬형 회피 구조 |

### 🍫 Stage 3 — 초콜릿 수집 (화이트 / 다크 초콜릿)

**난이도:** 중간

**목표:** 초콜릿 보스의 공격을 피해 초콜릿 조각을 모으기

**클리어 조건:** 일정 개수 이상의 초콜릿 조각 획득, 자동으로 지도를 펼쳐 다음 스테이지를 선택

**핵심 메커니즘:** 보스가 일정 시간동안 유지되는 통과 불가능한 초콜릿 장애물 생성. 초콜릿 장애물이 사라진 자리에 초콜릿 조각이 생기며, 플레이어는 떨어지는 장애물을 피하며 그 조각을 모은다. 만약 장애물이 조각 위에 다시 떨어지면 조각은 사라진다.

**게임플레이:** 체스판처럼 생긴 스테이지에 입장하면 초콜릿 보스가 화를 내며 하늘에서 테트리스 조각 모양의 판 초콜릿 장애물을 떨어뜨린다. 플레이어는 공주를 조작해 떨어지는 장애물을 피해야 하며, 장애물이 사라진 자리에 생성되는 초콜릿 조각에 접촉해 일정 개수 이상 모아야 한다.

**적/위험 요소:** 플레이어는 4칸의 체력을 지닌다. 초콜릿 장애물은 통과할 수 없으며 접촉해도 피해를 입지 않지만, 떨어지는 초콜릿 장애물에 깔릴 시 체력 1칸과 일정량의 초콜릿 조각을 잃는다.

**초콜릿 보스:** 왕관을 쓴 거대한 슬라임 형태다. 다크초콜릿을 선택하면 짙은 갈색 몸체를 가지며 떨어뜨리는 장애물도 갈색이다. 화이트초콜릿을 선택하면 흰색 몸체를 가지며 떨어뜨리는 장애물도 흰색이다.

| **요약** |  |
| --- | --- |
| **핵심 메커닉** | 보스가 일정 간격으로 맵에 초콜릿 조각을 생성 (장판 피하기 게임) |
| **보상 구조** | 괴물이 녹아 사라질 때 초콜릿 조각이 생성됨  |
| **위험 요소** | 떨어지는 장애물에 닿을 경우 체력 -1 및 누적 초콜릿 일부 손실 |
| **보스 설정** | 시간에 따라 흘러내리는 초콜릿 괴물 |
| **레퍼런스** | 슈퍼 마리오 시리즈 - 꽈당꽈당: 상하 충돌형 장애물 패턴 |

### 🍓 Stage 4 — 과일 채집 (사과 / 딸기 나무)

**난이도:** 높음

**목표:** 가시나무 보스의 공격을 피하며 과일을 전부 수확

**클리어 조건:** 주변의 나무와 상호작용해 일정 개수 이상의 열매 획득

**핵심 메커니즘:** 가시나무 보스가 일정 시간동안 유지되는 통과 가능한 가시덩굴 장애물을 생성. 화면 좌우에 배치된 4개의 과일나무 오브젝트와 상호작용 시 열매 획득. 일정 횟수 이상 상호작용한 과일나무 오브젝트는 소멸하며, 클리어를 위해서는 모든 과일나무와 상호작용해 소멸시켜야 한다.

**게임플레이:** 격자가 그려진 숲속 공터처럼 생긴 스테이지에 입장하면 화면 상단의 거대한 가시나무 보스가 눈을 뜨며 가시덩굴을 소환하기 시작한다. 플레이어는 공주를 조작해 가시덩굴을 피하며 화면 좌우 테두리의 과일나무와 일정 횟수 상호작용해 과일을 전부 모아야 한다.

**적/위험 요소:** 플레이어는 4칸의 체력을 지닌다. 가시덩굴 장애물은 통과 가능하나 접촉 시 매 초 체력 1칸을 잃는다.

**가시나무 보스:** 큰 나무줄기에 눈과 입, 수염이 달린 모습이다. 딸기를 선택하면 녹색 가시덩굴을 소환하며 주변 과일나무에 딸기가 달린다. 사과를 선택하면 갈색 가시덩굴을 소환하며 주변 과일나무에 사과가 달린다.

| **요약** |  |
| --- | --- |
| **핵심 메커닉** | 통과 가능한 가시덩굴 장애물 생성 (접촉 시 피해 있음) |
| **진행 구조** | 화면 양 옆의 과일나무 4개와 상호작용 → 일정 횟수 상호작용 후 소멸 |
| **클리어 조건** | 모든 과일 채집 = 모든 과일나무 소멸 |
| **보스 설정** | 얼굴이 있는 거대 가시나무. 선택 재료에 따라 덩굴 색상과 과일 변화 |
| **레퍼런스** | 헬테이커 - 저지먼트전 : 빠른 템포 회피 + 타이밍 액션 |

| 요소 | 구현 방식 | 설명 |
| --- | --- | --- |
| **카메라 제어** | `CinemachineVirtualCamera` 다중 설정 | 스테이지 오브젝트별 카메라 시점 전환 |
| **카메라 이벤트 트리거** | Timeline + Trigger Collider | 오브젝트 접근 시 카메라 자동 전환 |

| **전환 효과** | Fade In/Out (CanvasGroup) | 씬 간 연출 전환 부드럽게 처리 |
